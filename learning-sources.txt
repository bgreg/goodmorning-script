# Daily Learning Sources
#
# Format:
#   [Category Name]
#   sitemap:https://example.com/sitemap.xml
#   or
#   URL|Title
#
# Sitemap entries will fetch the sitemap and randomly select a page
# Static URL entries work as before
#
# HOW TO FIND SITEMAPS:
#
# 1. Check robots.txt:
#    Visit https://example.com/robots.txt and look for "Sitemap:" entries
#    Example: https://guides.rubyonrails.org/robots.txt
#
# 2. Common sitemap locations:
#    - /sitemap.xml (most common)
#    - /sitemap_index.xml (for large sites)
#    - /sitemap.xml.gz (compressed)
#    - /sitemaps/sitemap.xml
#
# 3. View page source:
#    Look for <link rel="sitemap"> tags in the HTML
#
# 4. Developer tools:
#    Check Network tab for sitemap requests when browsing docs
#
# 5. If no sitemap exists:
#    Use static URLs instead - just add URL|Title entries
